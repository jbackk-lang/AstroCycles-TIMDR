# Changelog
All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog  
and this project adheres to Semantic Versioning.

---

## [0.1.0] - 2026-06-07
### Added
- Initial project structure
- Planetary cycles dataset (`planetary_cycles.json`)
- Harmonic aspects dataset (`harmonic_aspects.json`)
- Harmonic generator algorithm
- Resonance detector algorithm
- Unified TIMDR pipeline
- ASCII resonance plot visualization
- ASCII harmonic diagram
- Example usage script (`examples/basic_usage.py`)
- Unit tests for harmonic generator and resonance detector
- Paper draft (`paper/paper_draft.md`)
- Roadmap (`ROADMAP.md`)
- MIT License
- `setup.py` for package installation

### Notes
This is the first structured release of the AstroCycles‑TIMDR framework.
It establishes the foundation for future scientific development,
visualization, and publication.
