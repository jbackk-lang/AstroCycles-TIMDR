# Harmonic Structure Diagram (TIMDR)

This document describes the base geometric structure used to represent
astrological harmonic cycles in the TIMDR framework.

## Core Elements

- **Cycle Ring**  
  A circular representation of a single astrological cycle.

- **Resonance Nodes**  
  Discrete points on the ring where transitions occur.

- **Harmonic Layers**  
  Concentric rings representing higher-order cycles.

- **Phase Shift Vector**  
  A directional marker showing the current position within the cycle.

Each dot represents a resonance node.  
Each layer represents a harmonic level.  
The number of nodes increases with harmonic complexity.
