# ASCII Harmonic Diagram (TIMDR)

This is a lightweight ASCII representation of the harmonic structure
used in the TIMDR framework. It is not a final diagram, but a conceptual
layout for documentation and algorithm development.

## Harmonic Layers (ASCII)

Layer 3:
●───●───●───●───●───●

Layer 2:
  ●────●────●────●

Layer 1:
    ●────────●

## Interpretation

- Each dot (●) represents a resonance node.
- Horizontal spacing represents phase distance.
- Higher layers contain more nodes and shorter intervals.
- The structure expands outward with each harmonic level.

## Purpose

This ASCII diagram is used for:
- documentation,
- conceptual explanation,
- early-stage visualization,
- and testing harmonic algorithms.

A graphical version will be added later.
