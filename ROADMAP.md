# AstroCycles-TIMDR — Project Roadmap (2026)

This roadmap outlines the planned development of the TIMDR-based
geometric cycle analysis framework.

---

## ✔ Phase 1 — Foundation (Completed)
- Core datasets:
  - Planetary cycles (JSON)
  - Harmonic aspect geometry (JSON)
- Core algorithms:
  - Harmonic layer generator
  - Resonance detector
- Pipeline:
  - Unified data → algorithm → output flow
- Documentation:
  - README structure
  - ASCII diagrams
  - Paper draft
- Visualization:
  - ASCII resonance plot
- Testing:
  - Initial unit tests
- Licensing:
  - MIT License added

---

## ▶ Phase 2 — Expansion (In Progress)
### 2.1 Data Expansion
- Add extended planetary datasets (retrograde cycles, inclinations)
- Add harmonic families (5th, 7th, 9th harmonics)
- Add resonance thresholds per harmonic level

### 2.2 Algorithmic Development
- Implement TIMDR phase operator
- Add multi-layer resonance mapping
- Add harmonic interference model
- Add cycle compression / normalization functions

### 2.3 Visualization
- Add multi-line ASCII diagrams
- Add timeline-style resonance plots
- Prepare structure for graphical diagrams (later)

---

## ▶ Phase 3 — Scientific Layer (Planned)
### 3.1 Paper Development

- Add figures (ASCII + later graphical)

### 3.2 Validation

- Evaluate harmonic density patterns

## Long-Term Vision
AstroCycles-TIMDR aims to become a modular, geometric framework for
analyzing periodic systems using harmonic structures and resonance
dynamics. The project will evolve toward:

- multi-layer harmonic modeling,  
- geometric cycle visualization,  
- scientific publication,  
- and open-source research tools.
